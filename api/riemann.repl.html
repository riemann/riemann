riemann.repl documentation

start-server
(start-server opts)
Starts a new REPL server, when one isn't already running.

start-server!
(start-server! opts)
Starts a new repl server. Stops the old server first, if any. Options:

:host (default "127.0.0.1")
:port (default 5557)
stop-server!
(stop-server!)
Stops the REPL server.
