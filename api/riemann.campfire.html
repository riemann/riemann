riemann.campfire

Forwards events to Campfire

campfire

(campfire token ssl sub-domain room-name)

Creates an adaptor to forward events to campfire. The campfire event will contain the host, state, service, metric and description.

Tested with: (streams (by [:host, :service] (let [camp (campfire “token”, true, “sub-domain”, “room”)] camp)))
